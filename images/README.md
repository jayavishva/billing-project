# Menu Item Images

Please add the following image files to this folder:

- `idly.jpg` - Image for Idly
- `puri.jpg` - Image for Puri  
- `vada.jpg` - Image for Vada
- `coffee.jpg` - Image for Coffee
- `pazhampori.jpg` - Image for Pazhampori

The website will display placeholder images if these files are not found. You can replace them with actual food images later.

Image recommendations:
- Format: JPG or PNG
- Size: 200x150 pixels or larger (will be automatically resized)
- Aspect ratio: 4:3 works best
